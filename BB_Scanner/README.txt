   BETBOOM LIVE SCANNER - Инструкция по запуску
═══════════════════════════════════════════════════════════════

БЫСТРЫЙ ЗАПУСК:
───────────────
1. Дважды кликните на файл "ЗАПУСК.bat"
2. При первом запуске установятся зависимости (может занять 2-5 минут)
3. Откроется окно сканера

ТРЕБОВАНИЯ:
───────────
- Windows 10/11
- Python 3.10 или выше
- Интернет-соединение

УСТАНОВКА PYTHON (если не установлен):
────────────────────────────────────────
1. Скачайте Python с https://www.python.org/downloads/
2. При установке ОБЯЗАТЕЛЬНО поставьте галочку "Add Python to PATH"
3. Перезапустите компьютер
4. Запустите "ЗАПУСК.bat" снова

ЧТО ДЕЛАЕТ СКАНЕР:
──────────────────
Сканирует BetBoom LIVE матчи по настольному теннису и ищет:
- TYPE A: Фаворит ведёт 2:0, кф на 3-й сет >= 1.90
- TYPE B: Фаворит ведёт 1:0 и лидирует во 2-м сете, кф >= 1.90
- TYPE C: Подозрительно равная линия (1.75-2.05) при доминировании

Все сигналы сохраняются в папке "app_data":
- signals.csv - все сигналы
- signals.sqlite - база данных
- errors.log - логи ошибок

ГОРЯЧИЕ КЛАВИШИ:
────────────────
- F5 - Обновить список матчей
- ESC - Закрыть приложение

ПРОБЛЕМЫ?
─────────
Если не запускается:
1. Проверьте, что Python установлен: откройте командную строку и введите "python --version"
2. Если Python не найден, установите его (см. выше)
3. Если ошибка при установке зависимостей, попробуйте:
   - Закрыть антивирус
   - Запустить "ЗАПУСК.bat" от имени администратора
